# 📖 Overview

This repository is part of my journey toward becoming an **AI-Powered Full-Stack Developer**.

Instead of only learning JavaScript concepts theoretically, I'm building a structured **DOM Engineering Lab** where each file focuses on a core DOM manipulation concept.

The goal is to develop a deep understanding of how browsers render, manipulate, and respond to user interactions—knowledge that serves as the foundation for modern frontend frameworks like React, Vue, Angular, and future AI-powered web applications.

# 📚 Topics Covered

## 01 — Selecting Elements

Learn different techniques for selecting DOM elements efficiently.

- getElementById()
- querySelector()
- querySelectorAll()
- getElementsByClassName()
- getElementsByTagName()

---

## 02 — Reading Information

Access and inspect DOM properties.

Examples include:

- textContent
- innerText
- innerHTML
- value
- id
- className
- classList

---

## 03 — Changing Content

Modify existing DOM elements dynamically.

Topics include:

- Updating text
- Updating HTML
- Changing values
- Rendering dynamic content

---

## 04 — Styling Elements

Manipulate styles using JavaScript.

Examples:

- Colors
- Font sizes
- Borders
- Display
- Visibility
- Dimensions

---

## 05 — Class Manipulation

Working with CSS classes.

Examples:

- add()
- remove()
- toggle()
- contains()
- replace()

---

## 06 — Attributes

Reading and modifying HTML attributes.

Examples:

- src
- href
- alt
- title
- data attributes

---

## 07 — Traversing the DOM

Understanding relationships between elements.

Examples:

- parentElement
- children
- firstElementChild
- lastElementChild
- nextElementSibling
- previousElementSibling

---

## 08 — Creating Elements

Generate new DOM elements dynamically.

Examples:

- createElement()
- append()
- appendChild()
- prepend()

---

## 09 — Removing & Replacing Elements

Modify the DOM structure.

Examples:

- remove()
- removeChild()
- replaceChild()
- replaceWith()

---

## 10 — Events

Building interactive applications through user interaction.

Covered topics:

- Mouse Events
- Keyboard Events
- Input Events
- Change Events
- Form Validation
- Real-Time Validation
- Event Listeners
- Event Objects

# 💡 Key Takeaways

Through this engineering lab I discovered that:

> **The DOM is not just a tree of HTML elements—it is the communication layer between JavaScript and the browser.**

Understanding the DOM is essential before working with frontend frameworks because frameworks abstract these same concepts rather than replacing them.
